This directory contains the necessary information to replicate the paper "Evaluating the Relationship of Software
Engineering Practices on Code Smells in Python-based ML Project" authored by Giammaria Giordano, Antonio Della Porta, Filomena Ferrucci, and Fabio Palomba.
Structure:
statistical_desriptive.R is a R script used to built the statistical descriptive of the NICHE dataset.
RQ1 folder contains all the scripts used to analzye the first research question i.e., the scripts used to build the statistical model and the script used to plot our results.
RQ2 fonder contains all the material used to respond on the second research question, i.e., the statistical models applied and the dataset splitted according to the LOC percentile (small, medium, and large).
